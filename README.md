# Deskripsi Repository

Repository ini dibuat sebagai bagian dari tugas Praktikum Rekayasa Perangkat Lunak (RPL) Pertemuan 12 dan 13. Repository ini berisi dokumentasi penggunaan GitHub sebagai media Version Control System (VCS), penyusunan Software Maintenance Plan, serta dokumentasi Software Architecture berdasarkan studi kasus sebuah Sistem E-Commerce.

Melalui praktikum ini mahasiswa diharapkan mampu memahami proses pengelolaan perubahan perangkat lunak menggunakan Git dan GitHub, membuat rencana pemeliharaan perangkat lunak, serta mendokumentasikan arsitektur perangkat lunak secara sistematis.

Adapun tujuan dari praktikum ini adalah sebagai berikut.
